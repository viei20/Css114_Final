# โปรแกรมคำนวณ Eigenvalue, Eigenvector และ Diagonalization

โปรแกรมนี้พัฒนาด้วยภาษา Python
ใช้สำหรับหาค่าเจาะจง (Eigenvalue) เวกเตอร์เจาะจง (Eigenvector) และการแปลงเป็นเมทริกซ์ทแยงมุม (Diagonalization)

---

## วิธีที่รองรับในโปรแกรม

1. การหารากของสมการลักษณะเฉพาะ (Characteristic Equation) สำหรับ 2x2 และ 3x3
2. การหาผลเฉลยสมการพหุนามกำลังสามด้วยวิธีของคาร์ดาน (Cardano's method)
3. การทำ Row Reduced Echelon Form (RREF)
4. การหา Null Space เพื่อสร้างเวกเตอร์เจาะจงที่เป็นอิสระเชิงเส้น
5. การหา Inverse Matrix ด้วยวิธี Augmented Matrix
6. การคำนวณคูณเมทริกซ์ (Matrix Multiplication)

โปรแกรมสามารถตรวจสอบได้ว่า

* ค่าเจาะจงเป็นจำนวนจริงหรือจำนวนเชิงซ้อน
* ระบบมีเวกเตอร์เจาะจงที่เป็นอิสระเชิงเส้นครบตามจำนวนมิติหรือไม่
* เมทริกซ์สามารถแปลงเป็นเมทริกซ์ทแยงมุมได้ (Diagonalizable) หรือไม่

---

## โปรแกรมสามารถทำอะไรได้บ้าง

* รับค่าสมาชิกของ Matrix A ขนาด 2x2 หรือ 3x3 จากผู้ใช้
* แสดงค่าเจาะจง ($\lambda$) พร้อมระบุ Algebraic Multiplicity
* แสดงเวกเตอร์เจาะจง ($v$) สำหรับแต่ละค่าเจาะจง
* ตรวจสอบและอธิบายเหตุผลว่าเมทริกซ์นั้น Diagonalizable หรือไม่
* สร้างและแสดงเมทริกซ์ $P$ (Eigenvector Matrix) และ $P^{-1}$
* แสดงเมทริกซ์ $D$ จากการพิสูจน์ด้วยสูตร $D = P^{-1}AP$

## ผลลัพธ์ที่โปรแกรมแสดง

### ส่วนที่ 1: ค่าเจาะจง (Eigenvalues)

โปรแกรมจะแสดงค่า $\lambda$ ที่คำนวณได้ พร้อมระบุ Algebraic Multiplicity ของแต่ละค่า (รองรับผลลัพธ์ที่เป็นจำนวนเชิงซ้อน `j`)

### ส่วนที่ 2: เวกเตอร์เจาะจง (Eigenvectors)

โปรแกรมจะแสดงเวกเตอร์ที่เป็นฐานหลัก (Basis) ของ Null Space สำหรับค่า $\lambda$ แต่ละตัว

### กรณีที่สามารถแปลงเป็นเมทริกซ์ทแยงมุมได้ (Diagonalizable)

* โปรแกรมจะแสดงข้อความ "✅ สามารถแปลงเป็นเมทริกซ์ทแยงมุมได้"
* แสดงเมทริกซ์ $P$ ซึ่งประกอบด้วยคอลัมน์ของเวกเตอร์เจาะจง
* แสดงเมทริกซ์อินเวอร์ส $P^{-1}$
* แสดงเมทริกซ์ทแยงมุม $D$ เพื่อพิสูจน์ความถูกต้องจากการคำนวณ $P^{-1}AP$

### กรณีที่ไม่สามารถแปลงเป็นเมทริกซ์ทแยงมุมได้ (Not Diagonalizable)

* โปรแกรมจะแสดงข้อความ "❌ ไม่สามารถแปลงเป็นเมทริกซ์ทแยงมุมได้"
* แจ้งสาเหตุเนื่องจากจำนวนเวกเตอร์เจาะจงที่เป็นอิสระเชิงเส้นมีไม่ครบตามขนาดของเมทริกซ์ (Geometric Multiplicity < Algebraic Multiplicity)
